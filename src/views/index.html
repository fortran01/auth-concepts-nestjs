Securing a Backend API with Auth0

This demo shows how to secure a backend API with Auth0 JWT tokens. It demonstrates:

  • Public and protected API endpoints
  • JWT token validation using Auth0's JWKS endpoint
  • Machine-to-machine (M2M) authentication for API access
  • Extracting user information from tokens

The demo provides:

  • A public API endpoint accessible without authentication
  • Protected API endpoints that require a valid Auth0 token
  • A test script to demonstrate the authentication flow
  • A demo UI to try it in your browser
Try Auth0 API Demo
Note: You'll need to configure Auth0 API settings in your .env file.
